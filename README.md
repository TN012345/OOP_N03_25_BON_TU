---

## 🚀 Cách chạy
### Dùng Terminal
```bash
cd src
javac *.java ../test/*.java ../review/*.java
java App
```

### Dùng IntelliJ IDEA hoặc VS Code
- Import thư mục `pure_java_project`
- Đánh dấu `src` là **Sources Root**, `test` là **Test Sources Root**
- Run file `App.java`
